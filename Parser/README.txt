Parses a list of names and returns a json object of the list

You can for example iput a list of things from the internett and get a fully usable json object out.

the executable file is compiled by pyinstaller, wich is a program that can make executable files of your python files.

----------------------------------------------------------------------------------------------------

to run program, put the list of things that you want to convert to json a json object into the names.txt file
after running the executable, the json object gets created in the new.txt file.

når du kjører parser.exe popper det opp en "new" fil som inneholder json objektet
brukte pyinstaller til å lage exe filen -- genialt
